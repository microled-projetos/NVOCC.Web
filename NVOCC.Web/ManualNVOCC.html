Cadastro de Grupos de Usuarios

No cadastro de grupos de usuarios é possível grupos de usuarios onde estes terão permissoes personalizadas.

Cadastro de Permissoes

No cadastro de permissoes é possível conceder e retirar permissoes pra grupos de usuarios criados no modulo anterir.

Cadastro de Usuarios

No cadastro de usuarios é possível efetivar o cadastro de todos os usuarios que terão acesso ao sistema, vinculando estes a grupos de acessos com suas devidas permissoes.

Cadastro de Parceiros

No cadastro de parceiros é possível efetivar o cadastro de todos os intervenientes do processo do agenciamento, seja ele utilizado nas etapas comerciais, operacionais ou financeiras.

Cambio FCA

No cadastro Cambio FCA cadastrar valor de cambio para moedas utilizadas pela FCA.

Cambio Parceiros

No cadastro Cambio Parceiros cadastrar valor de cambio para moedas utilizadas pelo Armador.

Itens Despesa

No cadastro ItensDespesa é possivel cadastrar taxas que serão utilizadas nos demais modulos do sistema.

Cadastro de Tabela de Frete

No cadastro da tabela de frete é possível efetivar o cadastro de todas as tabelas de frete dos Armadores/Cias Aéreas nos quais a empresa FCA LOG realizar fechamentos.

Taxas Locais do Armador

O cadastro de taxas locais será utilizado para realizar o cadastro de todas as taxas locais vinculadas a determinado armador, o botão para realizar o cadastro das taxas locais estará disponível na tela de cadastro de parceiro sempre que o flag transportador estiver ativo para o cadastro em edição.

Cadastro de Cotação

No cadastro de cotação é possível efetivar o cadastro de todas as cotações que serão destinadas aos clientes da FCA LOG, o módulo do cadastro em questão será separado em abas: - Informações Básicas: Possibilita o cadastramento do cabeçalho da cotação, nessa tela será informado dados como cliente, agente, serviço, vendedor etc. - Mercadoria/Frete/Taxas: Possibilita o cadastramento/edição de mercadorias, frete importado do tarifário e valores cadastrados na cotação.

Texto Cotacao

Tela para cadastro de texto padrão da impressão da cotação

Comissoes

ESTE MÓDULO CONTEMPLA: • Comissoes de Vendedor • Comissoes de Indicador Nacional • Comissoes de Indicador Internacional

MÓDULO OPERACIONAL

ESTE MÓDULO CONTEMPLA: • Visualização de todos os processos ativos da FCA LOG; • Inserção, edição e remoção de registros de Instrução de Embarque, Master e House. • Inserção, edição e remoção de registros de Taxas, Contêineres e Cargas.

MÓDULO FINANCEIRO

CONTAS A PAGAR Este módulo será dividido em três partes: • Solicitação de Pagamento • Montagem do Pagamento • Baixas/Cancelamentos CONTAS A RECEBER Este módulo será dividido em três partes: • Calculo de Recebimento • Emissão de Nota de Débito • Baixas/Cancelamentos • Envio para o Faturamento

Account

Este módulo tem como objetivo, cadastrar e conferir as invoices emitidas pelos agentes internacionais, controlar as remessas de valores ao exterior.

Faturamento

Módulo direcionado ao faturamento, onde é possivel emitir Notas de Débito, Recibos de Pagamento, RPS, Notas Fiscais e Boletos.
